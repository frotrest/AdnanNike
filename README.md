# Project Overview
**AdnanNike** is a modern study project inspired by original Nike website. The project is designed as a learning pet project that demonstrates practical React patterns, modern Nike's style and pleasant and convenient design.

## Purpose
Offer a convenient, long-term type of shoes shop for people who want to wear fashionable and spectacular shoes.

Serve as a portfolio piece showing frontend architecture, stylish and unique design.

## Features
**Core Functionality**

Unique design: I figure out that you've never seen sites kind of that with unique and breathtaking design.

Animations on Scroll: elements annotated with data-animate are animated when they enter the viewport using IntersectionObserver.
